Problem 1: Euler's Method

Given: $\frac{dy}{dx} = f(x, y)$ with initial condition $f(2) = y_0$

Task: Use Euler's method with two equal steps to approximate $f(x)$ at a target value

What IS Euler's Method? (The Deep Understanding)

Imagine hiking in thick fog. You can only see the slope of the ground right where you're standing, but the path ahead is invisible. Euler's method says: "Walk in a straight line at the current slope for a small distance, check the new slope, then adjust."

The Core Idea

A differential equation tells you the slope at any point. If you move a tiny distance along that slope, you approximate the curve. The smaller your steps, the closer you stay to the true curve.

The Formula:

$$y_{n+1} = y_n + h \cdot f'(x_n, y_n)$$

Where:

  • $y_{n+1}$ = new y-value (where we're going)
  • $y_n$ = current y-value (where we are)
  • $h$ = step size (how far we walk)
  • $f'(x_n, y_n)$ = slope at current point (from the differential equation)

Solution Process:

Let's say we go from $x = 2$ to $x = 1$ in two steps:

$$h = \frac{1-2}{2} = -0.5$$

Step 1:

  • Start at: $(2, f(2))$
  • Calculate slope: $\frac{dy}{dx}\bigg|_{x=2}$
  • New point: $y_1 = f(2) + (-0.5) \cdot \text{(slope at } x=2)$
  • New x: $x_1 = 2 + (-0.5) = 1.5$

Step 2:

  • Start at: $(1.5, y_1)$
  • Calculate slope: $\frac{dy}{dx}\bigg|_{x=1.5}$
  • Final: $y_2 = y_1 + (-0.5) \cdot \text{(slope at } x=1.5)$
  • Final x: $x_2 = 1.5 + (-0.5) = 1$
Euler's method gives linear approximations of curved paths. Error accumulates with each step. Smaller steps = less error, but more computation required.
Problem 2: Solving $\frac{dy}{dx} = -2x^2$

Part (A): Find the General Solution

Recognizing the Type: This is separable because the right side has ONLY $x$ (no $y$). We can "separate" the variables.

Solution:

$$\frac{dy}{dx} = -2x^2$$ $$dy = -2x^2 \, dx \quad \text{(multiply both sides by } dx\text{)}$$ $$\int dy = \int -2x^2 \, dx \quad \text{(integrate both sides)}$$ $$y = -2 \cdot \frac{x^3}{3} + C$$ $$y = -\frac{2x^3}{3} + C$$

The Deeper Question: Why Can We "Multiply by $dx$"?

This is actually shorthand for a more rigorous process:

  1. $\frac{dy}{dx}$ is the derivative (a limit)
  2. When we "multiply by $dx$," we're using the Fundamental Theorem of Calculus
  3. More precisely: $\int \frac{dy}{dx} \, dx = \int -2x^2 \, dx$
  4. The left side simplifies to just $y$ (integration "undoes" differentiation)
What does "$+ C$" mean? There are infinitely many functions with derivative $-2x^2$. Each one is vertically shifted by some constant. $C$ represents ALL possible vertical shifts. This is the general solution — a family of curves.

Part (B): Find the Particular Solution with $f(1) = -1$

Visual Understanding: Imagine a stack of parallel curves (all shifted vertically). The initial condition is like saying "use the curve that goes through this specific point."

Solution:

$$y = -\frac{2x^3}{3} + C$$

Plug in $(1, -1)$:

$$-1 = -\frac{2(1)^3}{3} + C$$ $$-1 = -\frac{2}{3} + C$$ $$C = -1 + \frac{2}{3} = -\frac{3}{3} + \frac{2}{3} = -\frac{1}{3}$$

Particular Solution:

$$f(x) = -\frac{2x^3}{3} - \frac{1}{3}$$
Problem 3: $\frac{dy}{dx} = -\frac{2x^2}{y}$

This Changes Everything! Now $y$ appears in the equation.

Part (A): Tangent Line at $(1, -1)$

What You Need for a Tangent Line:
  1. A point: $(1, -1)$ ✓ (given)
  2. A slope: $\frac{dy}{dx}$ evaluated at that point

Finding the Slope:

$$\frac{dy}{dx} = -\frac{2x^2}{y}$$

At $(1, -1)$:

$$\frac{dy}{dx}\bigg|_{(1,-1)} = -\frac{2(1)^2}{-1} = -\frac{2}{-1} = 2$$

Tangent Line Equation:

Point-slope form: $y - y_1 = m(x - x_1)$

$$y - (-1) = 2(x - 1)$$ $$y + 1 = 2x - 2$$ $$y = 2x - 3$$
The differential equation is like a "slope calculator." Plug in ANY point $(x, y)$, and it tells you the slope there. The tangent line is just the line with that slope through that point.

Part (B): Find Particular Solution with $f(1) = -1$

Separation of Variables — The Real Process

When $y$ appears in the equation, we need to separate $x$ terms from $y$ terms:

Step-by-Step Separation:

$$\frac{dy}{dx} = -\frac{2x^2}{y}$$

Multiply both sides by $y$:

$$y \cdot \frac{dy}{dx} = -2x^2$$

Multiply both sides by $dx$ (integration shorthand):

$$y \, dy = -2x^2 \, dx$$

Integrate both sides:

$$\int y \, dy = \int -2x^2 \, dx$$ $$\frac{y^2}{2} = -\frac{2x^3}{3} + C$$

Multiply by 2:

$$y^2 = -\frac{4x^3}{3} + K \quad \text{(where } K = 2C\text{)}$$

Apply Initial Condition $f(1) = -1$:

$$(-1)^2 = -\frac{4(1)^3}{3} + K$$ $$1 = -\frac{4}{3} + K$$ $$K = 1 + \frac{4}{3} = \frac{3}{3} + \frac{4}{3} = \frac{7}{3}$$

So:

$$y^2 = -\frac{4x^3}{3} + \frac{7}{3} = \frac{7 - 4x^3}{3}$$

Solve for $y$:

$$y = \pm\sqrt{\frac{7 - 4x^3}{3}}$$

Since $f(1) = -1$ is negative, we use the negative root:

$$f(x) = -\sqrt{\frac{7 - 4x^3}{3}}$$

The Big Conceptual Leap

When you have $y$ in the differential equation:

  1. You integrate with respect to $y$ on one side
  2. This gives you $\frac{y^2}{2}$, not just $y$
  3. You get an implicit solution ($y^2 = \text{something}$)
  4. You must solve for $y$ explicitly
  5. The $\pm$ means two solutions — initial condition picks which one
Why does this work? $\int y \, dy$ means: "What function, when differentiated with respect to $y$, gives $y$?" Answer: $\frac{y^2}{2}$ (plus constant).
Problem 4: Slope Fields for $\frac{dy}{dx} = \frac{y-1}{x+y}$

Part (A): Sketch the Slope Field

Understanding Slope Fields Conceptually

A slope field is a visual representation of ALL possible solutions at once.

At each grid point $(x, y)$:

  • Calculate $\frac{dy}{dx} = \frac{y-1}{x+y}$
  • Draw a tiny line segment with that slope
  • Don't connect the segments!
Strategic Approach: Instead of calculating 50 slopes, look for PATTERNS!

Find Key Features:

1. Where is slope ZERO (horizontal segments)?

$$\frac{y-1}{x+y} = 0 \implies y - 1 = 0 \implies y = 1$$

Along the line $y = 1$, all slopes are horizontal.

2. Where is slope UNDEFINED (vertical segments)?

Division by zero when:

$$x + y = 0 \implies y = -x$$

Along the line $y = -x$, slopes are vertical (undefined).

3. Where is slope = 1 (diagonal at 45°)?

$$\frac{y-1}{x+y} = 1$$ $$y - 1 = x + y$$ $$-1 = x \implies x = -1$$

Along the line $x = -1$, all slopes equal 1.

Sample Calculations (for verification):

  • $(0, 0)$: slope $= \frac{0-1}{0+0} = \frac{-1}{0}$ = undefined ✓ (on $y=-x$ line)
  • $(0, 2)$: slope $= \frac{2-1}{0+2} = \frac{1}{2}$
  • $(1, 1)$: slope $= \frac{1-1}{1+1} = 0$ ✓ (on $y=1$ line)
  • $(-1, 0)$: slope $= \frac{0-1}{-1+0} = 1$ ✓ (on $x=-1$ line)
The slope field shows the "flow" of the differential equation. Solution curves must be tangent to the field at EVERY point. You can visualize solutions by following the flow with your eye.

Part (B): Find Horizontal Line $y = c$

Translation: Find where $\frac{dy}{dx} = 0$ for ALL $x$.
$$\frac{y-1}{x+y} = 0$$

For a fraction to equal zero, the numerator must be zero:

$$y - 1 = 0 \implies y = 1$$

Answer: $c = 1$

Deeper Meaning: Equilibrium Solutions

This is an equilibrium solution (also called a particular solution):

  • If $y$ starts at 1, then $\frac{dy}{dx} = 0$, so $y$ never changes
  • It's a stable state
  • Also called a "constant solution"

Why is this special? Most solutions are CURVES. This solution is a straight horizontal line — nothing ever changes along it.

Problem 6: Logistic Growth Model

Given: $\frac{dP}{dt} = \frac{2}{5000}P(5000 - P)$ with $P(0) = 3000$

Understanding Logistic Growth DEEPLY

Why this equation exists: Real populations don't grow exponentially forever. They hit limits (food, space, resources). The logistic model captures this reality.

Breaking Down the Equation:

$$\frac{dP}{dt} = \underbrace{\frac{2}{5000}P}_{\text{growth rate}} \cdot \underbrace{(5000 - P)}_{\text{limiting factor}}$$

What Each Part Means:

Component Meaning
$P$ term Growth proportional to population (more individuals → more reproduction)
$(5000 - P)$ term When $P$ is small: $(5000-P) \approx 5000$ (almost exponential)
When $P \to 5000$: $(5000-P) \to 0$ (growth stops)
When $P > 5000$: $(5000-P) < 0$ (population decreases!)
$K = 5000$ Carrying capacity (maximum sustainable population)

Behavior Analysis:

  • $P = 0$: $\frac{dP}{dt} = 0$ (no population, no growth)
  • $0 < P < 5000$: $\frac{dP}{dt} > 0$ (growing)
  • $P = 5000$: $\frac{dP}{dt} = 0$ (equilibrium reached)
  • $P > 5000$: $\frac{dP}{dt} < 0$ (declining back to 5000)

The Solution (General Form):

$$P(t) = \frac{K}{1 + Ae^{-rt}}$$

Where:

  • $K$ = carrying capacity (5000)
  • $r$ = growth rate (extracted from coefficient)
  • $A$ = determined by initial condition $P(0) = 3000$
Conceptual Beauty: The graph of $P(t)$ is an S-curve:
  • Starts slow (few individuals)
  • Accelerates (exponential-like growth)
  • Levels off (approaching carrying capacity)
This models: bacterial growth, viral spread, technology adoption, rumor spreading!
Problem 7: Particle Motion — The Integration Chain

Given:

  • Acceleration: $a(t) = \frac{1}{\sqrt{t}}$ for $t > 0$
  • Velocity at $t = 4$: $v(4) = -0.1$ m/s
  • Position at $t = 4$: $h(4) = $ [value from worksheet]
  • Initial position: $h(0) = 8$

Understanding the Position-Velocity-Acceleration Relationship

The Derivative Chain:

Position $h(t)$ $\xrightarrow{\text{differentiate}}$ Velocity $v(t) = h'(t)$ $\xrightarrow{\text{differentiate}}$ Acceleration $a(t) = v'(t) = h''(t)$

To solve, we INTEGRATE backwards:

Acceleration $a(t)$ $\xleftarrow{\text{integrate}}$ Velocity $v(t)$ $\xleftarrow{\text{integrate}}$ Position $h(t)$

Step 1: Find Velocity $v(t)$

$$a(t) = \frac{1}{\sqrt{t}} = t^{-1/2}$$ $$v(t) = \int t^{-1/2} \, dt$$

Using power rule: $\int t^n \, dt = \frac{t^{n+1}}{n+1} + C$

$$v(t) = \frac{t^{1/2}}{1/2} + C_1 = 2t^{1/2} + C_1 = 2\sqrt{t} + C_1$$

Apply condition $v(4) = -0.1$:

$$-0.1 = 2\sqrt{4} + C_1$$ $$-0.1 = 2(2) + C_1$$ $$-0.1 = 4 + C_1$$ $$C_1 = -4.1$$

Therefore: $v(t) = 2\sqrt{t} - 4.1$

Step 2: Find Position $h(t)$

$$v(t) = 2\sqrt{t} - 4.1 = 2t^{1/2} - 4.1$$ $$h(t) = \int (2t^{1/2} - 4.1) \, dt$$ $$h(t) = 2 \cdot \frac{t^{3/2}}{3/2} - 4.1t + C_2$$ $$h(t) = 2 \cdot \frac{2}{3}t^{3/2} - 4.1t + C_2$$ $$h(t) = \frac{4}{3}t^{3/2} - 4.1t + C_2$$

Apply condition $h(4) = $ [value]:

$$[\text{value}] = \frac{4}{3}(4)^{3/2} - 4.1(4) + C_2$$ $$[\text{value}] = \frac{4}{3}(8) - 16.4 + C_2$$ $$[\text{value}] = \frac{32}{3} - 16.4 + C_2$$ $$[\text{value}] = 10.67 - 16.4 + C_2$$

Solve for $C_2$...

Then verify with $h(0) = 8$ (or use this as the condition instead).

The Deep Concept: Why Two Constants?

  • Each integration adds ONE arbitrary constant
  • We integrated TWICE ($a \to v \to h$), so TWO constants
  • We need TWO initial conditions to determine both constants

Physical Meaning:

  • $C_1$ in velocity: initial velocity (how fast you start)
  • $C_2$ in position: initial position (where you start)
Real-World Analogy: If I tell you a car's acceleration, you can't know where it is without knowing:
  1. How fast it was going initially (determines velocity constant)
  2. Where it started (determines position constant)
Problem 10: Slope Field Matching — $\frac{dP}{dt} = -6\cos(t)$

Part (A): Which Slope Field?

Analyze the Equation:

$$\frac{dP}{dt} = -6\cos(t)$$

Key Observation #1: Depends only on $t$, NOT on $P$

→ Slope is the SAME for all $P$ at a given $t$
→ Slopes should form HORIZONTAL BANDS (same slope in vertical columns)

Behavior with $t$:

$t$ value $\frac{dP}{dt}$ Slope Direction
$t = 0$ $-6\cos(0) = -6(1) = -6$ Negative, steep down
$t = \frac{\pi}{2}$ $-6\cos(\frac{\pi}{2}) = -6(0) = 0$ Horizontal
$t = \pi$ $-6\cos(\pi) = -6(-1) = 6$ Positive, steep up
$t = \frac{3\pi}{2}$ $-6\cos(\frac{3\pi}{2}) = -6(0) = 0$ Horizontal
$t = 2\pi$ $-6\cos(2\pi) = -6(1) = -6$ Negative (cycle repeats)

Look for a slope field with:

  • ✓ Vertical consistency (same slope in each column)
  • ✓ Oscillating pattern following $-\cos(t)$
  • ✓ Starting with negative slopes
  • ✓ Crossing zero at $t = \frac{\pi}{2}$

Part (B): Solve the Differential Equation

$$\frac{dP}{dt} = -6\cos(t)$$

Integrate both sides:

$$P(t) = \int -6\cos(t) \, dt$$ $$P(t) = -6\sin(t) + C$$
Graph Characteristics:
  • Sine wave (shifted/scaled)
  • Amplitude: 6
  • Starts at $P = C$ (when $t = 0$, since $\sin(0) = 0$)
  • Oscillates between $C - 6$ and $C + 6$

Key Conceptual Synthesis

1. Separation of Variables — The Deep Truth

When we "multiply by $dx$," we're really using:

$$\frac{dy}{dx} = f(x)g(y)$$

↓ (Rewrite using differentials)

$$\frac{1}{g(y)} dy = f(x) dx$$

↓ (Integrate both sides)

$$\int \frac{1}{g(y)} \, dy = \int f(x) \, dx$$

The left side integrates with respect to $y$, right with respect to $x$.

2. Initial Conditions — Why We Need Them

  • General solution: Has arbitrary constant(s) — represents a FAMILY of functions
  • Particular solution: Uses initial condition to pin down ONE specific function
  • Integration chain: $n$ integrations need $n$ conditions

3. Slope Fields — Visual Power

  • Shows ALL solutions simultaneously
  • Solution curves are always tangent to the field
  • Equilibrium solutions are where slope = 0 everywhere
  • Can predict solution behavior without solving

4. Euler's Method — Numerical Approximation

  • When you CAN'T solve exactly, approximate numerically
  • Trade-off: smaller steps = more accurate, more work
  • Error accumulates — each step compounds previous errors
  • Foundation of computational differential equations

5. Models in the Real World

Model Equation Application
Exponential $\frac{dP}{dt} = kP$ Unlimited growth
Logistic $\frac{dP}{dt} = kP(M-P)$ Growth with carrying capacity
Newton's Cooling $\frac{dT}{dt} = k(T-T_a)$ Temperature equilibration
Motion $a(t) \to v(t) \to s(t)$ Physics fundamentals

These equations MODEL reality — they're powerful because they describe CHANGE.

Practice Mindset: When you see a differential equation, ask yourself:
  1. What TYPE is this? (separable, logistic, linear, etc.)
  2. What does it MEAN? (What's changing? Why?)
  3. What METHOD do I use? (separate, integrate, substitute?)
  4. What will the SOLUTION look like? (oscillating, growing, stable?)
  5. How does the INITIAL CONDITION matter?

Don't just solve mechanically — understand the story the equation tells.
